
# Modern CV Builder

A modern, responsive CV (Resume) Builder web application that allows users to easily create, customize, 
and preview professional CVs in real time. This tool simplifies the process of building a clean and structured resume without needing design or technical skills.

---

## Features

* Live CV preview while typing
* Clean and modern UI design
* Responsive layout (works on desktop and mobile)
* Easy input for personal information, education, and experience
* Structured and professional CV format
* Fast and lightweight

## How It Works

1. Enter your personal details
2. Add your education and work experience
3. The CV updates automatically in real time
4. Preview your professional CV layout
5. Download file
